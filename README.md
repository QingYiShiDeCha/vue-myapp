# vue-myapp

## 说明
+ 基于 NodeJs 16版本以上创建的项目，请确保NodeJS版本在16以上

## npm set
```
npm config set registry https://registry.npmmirror.com/
```

### npm install yarn
```
npm install -g yarn
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
